# 🤖 AI (Artificial Intelligence)

### 📌 Why was it invented?

To build systems that can perform tasks requiring human intelligence.

### 🌍 Real-Life Example

A GPS that suggests the best route based on current traffic.

### ☁ Google Example

Google Assistant, Gemini, Google Translate.

### ⚡ One-Line Interview Answer

Artificial Intelligence enables computers to perform tasks that normally require human intelligence.

---

# 🧠 Machine Learning

### 📌 Why?

Instead of programming every rule manually, let computers learn from data.

### 🌍 Real-Life Example

Email spam detection that improves as it sees more emails.

### ☁ Google Example

YouTube recommendations, Google Photos categorization.

### ⚡ One-Line Interview Answer

Machine Learning allows systems to learn patterns from data instead of relying on manually written rules.

---

# 🧩 Large Language Models (LLMs)

### 📌 Why?

To understand and generate human language at scale.

### 🌍 Real-Life Example

A knowledgeable assistant that can answer questions, summarize text, and write code.

### ☁ Google Example

Gemini.

### ⚡ One-Line Interview Answer

An LLM is a neural network trained on massive amounts of text to understand and generate language.

---

# 🔍 Retrieval-Augmented Generation (RAG)

### 📌 Why?

LLMs shouldn't rely only on what they learned during training.

They should retrieve fresh, relevant information before answering.

### 🌍 Real-Life Example

Instead of answering from memory, a lawyer first checks the latest legal documents.

### ☁ Google Example

Enterprise search and document-based AI assistants.

### ⚡ One-Line Interview Answer

RAG combines information retrieval with an LLM to generate more accurate and up-to-date responses.

---

# 🧠 Embeddings

### 📌 Why?

Computers don't understand words—they understand numbers.

Embeddings convert text into numerical vectors that capture meaning.

### 🌍 Real-Life Example

Placing similar books close together in a library.

### ☁ Google Example

Semantic Search, recommendation systems.

### ⚡ One-Line Interview Answer

Embeddings represent text as vectors so machines can compare semantic similarity.

---

# 🗄 Vector Databases

### 📌 Why?

Traditional databases search by exact values.

AI applications need to search by meaning.

### 🌍 Real-Life Example

Finding books with similar ideas instead of the exact title.

### ☁ Google Example

Vertex AI Vector Search.

### ⚡ One-Line Interview Answer

A Vector Database stores embeddings and enables fast similarity search.

---

# 🤖 AI Agents

### 📌 Why?

Sometimes AI should not only answer questions but also perform actions.

### 🌍 Real-Life Example

A travel assistant that books flights, hotels, and adds events to your calendar.

### ☁ Google Example

Gemini-powered workflow automation.

### ⚡ One-Line Interview Answer

An AI Agent plans, reasons, and performs tasks using tools to achieve a goal.
